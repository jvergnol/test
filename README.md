# Prueba Tulen

_Pequeño proyecto que cuenta con un Login y un datatable_


### Componentes utilizados 📋

```
coreui
sweetalert2
sweetalert2-react-content
axios
mdbreact
moment

```

### Instalación 🔧

``` bash
# clone the repo
$ git clone https://github.com/jvergnol/test.git

# go into app's directory
$ cd my-project

# install app's dependencies
$ npm install
```
## Deployment 📦

``` bash
$ npm start
```
## Production 📦

``` bash
$ npm run build
```
